# Strapi Backend (v5)

This project is migrated to Strapi v5 structure.

## Quick start

```bash
nvm use 20
npm install
npm run build
npm run develop
```

## Environment

Use `.env.example` as base and configure SQL DB + Strapi secrets.

See `MIGRATION_V5.md` for important notes (especially MongoDB to PostgreSQL migration).
