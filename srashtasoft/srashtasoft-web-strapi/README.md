
# Strapi Backend 

This project contains the backend application of the project


## Steps to run project 

#### Build the docker 

```http
  docker build -t <name_of_container_user_wants> .
```

#### run docker

```http
  docker run  -p 1337:1337 <name_of_container_user_has_given_to_build> 
```


#### If Port error appears than please stop the container where the perticular port defined is running 


#### Install docker is pre-requirement of the project
