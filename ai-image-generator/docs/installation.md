# Installation Guide

## Requirements
- WordPress 5.0+
- PHP 7.4+
- cURL extension enabled
- API key from OpenAI or Stability AI

## Installation Steps

1. **Download** the plugin ZIP file
2. **Upload** via WordPress admin:
   - Go to Plugins > Add New
   - Click "Upload Plugin"
   - Choose the ZIP file
   - Click "Install Now"
3. **Activate** the plugin
4. **Configure** API keys in Settings > AI Image Generator

## API Key Setup

### OpenAI DALL-E
1. Visit https://platform.openai.com/
2. Create account and get API key
3. Add to plugin settings

### Stability AI
1. Visit https://platform.stability.ai/
2. Create account and get API key  
3. Add to plugin settings

## First Use
1. Edit any post/page
2. Find "AI Featured Image Generator" meta box
3. Enter image description
4. Click "Generate Featured Image"
