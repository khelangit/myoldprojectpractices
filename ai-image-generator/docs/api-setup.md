# API Configuration Guide

## Supported Providers

### DALL-E 3 (OpenAI)
- **Cost**: ~$0.04 per image
- **Sizes**: 1024x1024, 1792x1024, 1024x1792
- **Quality**: High quality, creative interpretations

### Stable Diffusion XL (Stability AI)  
- **Cost**: ~$0.018 per image
- **Sizes**: Multiple options available
- **Quality**: High detail, realistic images

## Rate Limiting
Configure in plugin settings to control costs:
- Default: 50 images per hour per user
- Adjustable based on your needs

## Cost Management
- Monitor usage in plugin dashboard
- Set reasonable rate limits
- Track spending per user
