Generate AI images directly in WordPress using DALL-E and Stable Diffusion APIs.

== Description ==

AI Image Generator allows you to create stunning images using artificial intelligence directly within your WordPress admin. Perfect for bloggers, content creators, and website owners who need high-quality images quickly.

**Key Features:**
* Generate images from text descriptions
* Support for DALL-E 3 and Stable Diffusion
* One-click featured image setting
* Multiple art styles and sizes
* Auto-compression and optimization
* Media library integration
* Usage tracking and cost monitoring

**Supported APIs:**
* OpenAI DALL-E 3
* Stability AI Stable Diffusion

== Installation ==

1. Upload the plugin files to `/wp-content/plugins/ai-image-generator/`
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Settings > AI Image Generator to configure your API keys
4. Start generating images in your post editor!

== Frequently Asked Questions ==

= Do I need API keys? =
Yes, you'll need either an OpenAI API key for DALL-E or a Stability AI key for Stable Diffusion.

= How much do API calls cost? =
DALL-E 3: ~$0.04 per image
Stable Diffusion: ~$0.018 per image
Costs are charged directly by the API providers.

= Can I use both APIs? =
Yes! You can switch between providers in the settings.
